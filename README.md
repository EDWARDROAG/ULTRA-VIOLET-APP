# Aura Violet

Sitio web para Aura Violet SAS, emprendimiento creativo de detalles personalizados, diseno grafico, trabajos escolares y papeleria artesanal.

## Tecnologias

- React 18
- Vite
- React Router
- TailwindCSS
- Framer Motion
- react-hook-form
- Sonner

## Ejecutar localmente

Instala dependencias desde la raiz del frontend:

```bash
npm install
```

Levanta la aplicacion:

```bash
npm run dev --prefix apps/web
```

Compila para produccion:

```bash
npm run build --prefix apps/web
```

## Despliegue

La app usa `HashRouter`, por lo que funciona en GitHub Pages sin configurar reglas de servidor para rutas internas.
